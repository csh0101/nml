# 标签: 段落计划

## 作用
把场景拆成可校验的段级执行序列。

## 父标签
- 场景

## 子标签
- 段（1..n）

## 属性
- 校验: `enum`，可选。`和等于场景字数目标 | 对齐场景配比`。
- 偏差上限: `percent`，可选。仅在 `校验=对齐场景配比` 时生效，默认 `15%`。
- 流转校验: `enum`，可选。`开启`。
- 状态校验: `enum`，可选。`开启`。
- 体力演化: `enum`，可选。`同级或下降`。
- 情绪演化: `enum`，可选。`受限迁移`。
- 关系校验: `enum`，可选。`开启`。
- 关系演化: `enum`，可选。`波动不超过1`。
- 关系类型演化: `enum`，可选。`严格一致 | 受限迁移`。

## 约束
- `校验=和等于场景字数目标` 时：所有 `段.字数目标` 之和必须等于 `场景.字数目标`。
- `校验=对齐场景配比` 时：要求场景存在 `配比.对话/动作`，并校验每段 `对话占比/动作占比` 偏差。
- `流转校验=开启` 时：按 `铺垫 -> 推进 -> 转折 -> 爆发 -> 回落 -> 钩子` 做顺序建议校验。
- `状态校验=开启` 时：基于相邻段的 `状态` 做连续性或演化模式校验。
- `关系校验=开启` 时：基于相邻段的 `关系状态` 做连续性或演化模式校验。
- 设置 `关系类型演化` 时，`关系状态.类型` 变为必填。
