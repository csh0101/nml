# NML 自检清单

1. 根节点是否为 `<小说项目>`，且有 `名称`、`版本`。
2. `元数据` 是否存在。
3. 每个 `章节` 是否至少一个 `场景`。
4. 每个 `场景` 是否恰好一个 `任务`。
5. `地点引用`、`视角角色`、`状态.角色`、`关系状态.角色/目标` 是否有定义。
6. 每个 `段` 是否具备 `字数目标/功能/焦点/节奏`。
7. 每个 `段` 是否至少一个非空 `信息点`，且 `类型/权重` 合法。
8. 若启用 `校验=和等于场景字数目标`，段字数和是否等于场景字数。
9. 若启用关系或状态演化模式，相关属性是否补全（如 `关系状态.类型`）。
10. 最后执行：
    - `npm run check -- <file.nml> --parser tree-sitter`
    - 可选：`npm run check -- <file.nml> --parser rust-nom`
