# Chapter-3000 自检清单

1. `章节蓝图.字数目标` 是否为 `3000`。
2. 所有 `场景.字数目标` 求和是否等于 `3000`。
3. 每个场景是否包含：`任务/信息揭示/段落计划/场景收束`。
4. 每个 `段落计划` 中 `段.字数目标` 求和是否等于对应 `场景.字数目标`。
5. 每个 `段` 是否有 `字数目标/功能/焦点/节奏`。
6. 每个 `段` 是否至少一个 `信息点`，且 `类型/权重` 合法。
7. 引用字段是否命中：`地点引用/视角角色/状态.角色/关系状态.角色/关系状态.目标`。
8. 是否存在 `必须提及` 与 `禁止提及` 重叠项。
9. 校验命令是否通过：
   - `npm run check -- <file.nml> --parser tree-sitter`
   - 可选 `npm run check -- <file.nml> --parser rust-nom`
